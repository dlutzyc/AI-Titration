本文件夹内包含有现有的部分滴定实验的权重文件

[Acid-Base_Methyl_Orange.pth](Acid-Base_Methyl_Orange.pth) 酸碱滴定实验，使用**盐酸**滴定**氢氧化钠**，指示剂为甲基橙。颜色分类为'yellow'和'orange'。

[Coordination_Eriochrome_Black_T.pth](Coordination_Eriochrome_Black_T.pth) 配位滴定实验，使用**EDTA**滴定**自来水**，指示剂为铬黑T。颜色分类为'yellow'和'orange'。

[Precipitation_K2CrO4.pth](Precipitation_K2CrO4.pth) 沉淀滴定实验，使用**硝酸银**滴定**氯化钠**，指示剂为铬酸钾。颜色分类为'yellow'和'orange'。
